# 🌐 Portfolio Website

A personal portfolio website showcasing my projects, skills, and experience. This website serves as a central place to highlight my work and provide an easy way for others to connect with me.

* Clean and modern UI ✨

* Contact section 📬

## ⚙️ How to Run Locally

1. Clone the repository:

```
git clone https://github.com/dhruv-narula2006/Portfolio_Website.git
```

2. Navigate to the folder:

```
cd Portfolio_Website
```

3. Open `index.html` in your browser.
